# Hotel-Reservation-System---Database-Project
This project contains a relational MySQL database designed to model core operations of a hotel. It includes a tables for rooms, amenities, guests, reservations, and the relationships between them. The goal is to represent how a hotel manages bookings, room features, and guest information using a structured schema.

![Alt text](ER-diagram.png)

## Purpose: demonstrate how to design and populate a hotel management database, focus on table structure that can be used for queries, and testing.

## Technology: MySQL, DataGrip
